# Planned Pin Assignments (Nano)

RC Receiver
* D7 - RC Ch1
* D8 - RC Ch2
* D9 - RC Ch3
* D10 - RC Ch4
* D11 - RC Ch5
* D12 - RC Ch6

PCA9685
* A4 (SDA)
* A5 (SCL)

DFPlayerMini
* D5 (TX)
* D6 (RX)

Volume Control
* A6
